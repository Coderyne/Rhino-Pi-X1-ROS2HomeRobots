# 轮腿式平衡机器人仿真

基于达妙开源轮腿机器人仿真项目，目录包含机器人的 Simulink 动力学模型及 LQR 控制器设计相关 MATLAB 脚本。

> 参考论文：**《轮腿式平衡机器人控制》** — 陈阳

---

## 文件说明

| 文件 | 类型 | 说明 |
|------|------|------|
| `blance_leg2.slx` | Simulink 模型 | 轮腿平衡机器人主仿真模型，集成了动力学模型和 LQR 控制器 |
| `get_k_length.m` | MATLAB 函数 | 根据腿长 `leg_length` 建立状态空间模型，求解 LQR 最优增益矩阵 `K`（2×6）。基于拉格朗日动力学推导，包含驱动轮、摆杆（腿）、机体三部分耦合模型 |
| `get_k.m` | MATLAB 脚本 | 批量调用 `get_k_length` 计算不同腿长（0.1~0.21m）下的 K 矩阵，进行三次多项式拟合，并输出 `fp32` 格式的 C 语言数组，供嵌入式端（STM32）直接使用 |
| `VMC_calc.m` | MATLAB 函数 | 虚拟模型控制（VMC）计算：给定期望力 `F0` 和髋关节扭矩 `Tp`，通过雅可比矩阵换算为上/下关节电机扭矩 `Tp1`、`Tp2` |
| `d_phi0.m` | MATLAB 脚本 | 符号计算：推导机体姿态（`L0`、`phi0`）对关节角度（`phi1`、`phi4`）的雅可比矩阵，用于运动学分析 |
| `v_angle_FixedTorque.m` | MATLAB 脚本 | 跳跃能力验证：在恒定扭矩下求解角度-角速度二阶微分方程，计算机架速度随大腿角度的变化曲线，判断是否满足最小起跳速度 |

## 使用方式

### 环境要求

- **MATLAB R2021a 或更高版本**（需 Simulink + Control System Toolbox + Symbolic Math Toolbox）

### 运行步骤

1. **生成 LQR 增益矩阵**
   ```matlab
   run('get_k.m')
   ```
   该脚本会：
   - 遍历不同腿长，调用 `get_k_length` 计算 LQR 增益 `K`
   - 对 K 矩阵的 12 个元素分别做三次多项式拟合
   - 在 Workspace 中生成默认腿长（0.14m）对应的 `K` 变量
   - 输出 `fp32` 格式的 C 数组系数，可直接复制到嵌入式代码

2. **打开 Simulink 模型仿真**
   ```matlab
   open_system('blance_leg2')
   ```
   确认 Workspace 中已有 `K` 矩阵后，运行仿真观察机器人平衡控制的动态响应。

3. **跳跃能力验证（可选）**
   ```matlab
   run('v_angle_FixedTorque.m')
   ```
   绘制机架速度-大腿角度曲线，验证扭矩是否满足起跳条件。

### 关键参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 驱动轮半径 `R` | 0.0625 m | — |
| 腿长范围 | 0.10 ~ 0.21 m | 可调 |
| 默认腿长 | 0.14 m | `get_k.m` 中设定 |
| 机体质量 `M` | 2.087 kg | — |
| 腿总质量 `mp` | 3.887 kg | 两条腿 |
| 轮总质量 `mw` | 1.210 kg | 两个驱动轮 |
| 机器人总重 | ~7.18 kg | — |

## 控制架构

```
期望状态 → [LQR 增益 K] → [轮扭矩 T, 髋扭矩 Tp] → 轮腿动力学模型 → 实际状态
                ↑                                              |
                └──────────── 状态反馈 ─────────────────────────┘
```

- **状态量**（6维）：`[theta, d_theta, x, d_x, phi, d_phi]` — 摆杆角/角速度、轮位移/速度、机体角/角速度
- **控制量**（2维）：`[T, Tp]` — 驱动轮扭矩、髋关节扭矩
- **LQR 加权**：Q 矩阵侧重速度跟踪和姿态稳定，R 矩阵限制控制输出幅值

## 模型说明

Simulink 模型 `blance_leg2.slx` 基于以下假设：

- 机器人简化为三刚体模型：**驱动轮** + **摆杆（腿）** + **机体**
- 驱动轮与地面间无滑动（纯滚动约束）
- 髋关节为理想旋转副，无摩擦
- 俯仰角（phi）和摆杆角（theta）均为小角度附近线性化

VMC（虚拟模型控制）通过运动学雅可比将任务空间力/力矩映射到关节空间力矩，相关推导见 `d_phi0.m`。
