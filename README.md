## 文件结构

整个机器人项目被分成如下的几个部分，分别位于仓库不同目录下，内部有更详细的说明，读者可以按需查看：

- [`Guide`](Guide)：项目指南，包含项目介绍、功能说明、使用教程等文档
- [`Machine`](Machine)：机器人机械结构设计，包含机械设计文件和装配说明
- [`Code`](Code)：机器人软件部分，包含上位机和下位机模块的代码工程
- [`PCB`](PCB)：机器人电路设计，包含电路原理图和PCB设计文件
- [`Matlab`](Matlab)：算法仿真，包含模型建立、算法设计和仿真文件等
