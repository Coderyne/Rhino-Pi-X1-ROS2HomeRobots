# 3D 打印件

机器人外壳和结构件的 3D 打印文件，可直接导入切片软件进行打印。

## 打印清单

| 零件名称 | 数量 | 文件名 |
| -------- | :--: | ------ |
| 前挡板 | 1 | `前挡板.STEP` |
| 后挡板 | 1 | `后挡板.STEP` |
| 轮毂打印件-内圈 | 2 | `轮毂打印件-内圈.STEP` |
| 上握把 | 1 | `上握把.STEP` |
| 深度相机支架 | 1 | `深度相机支架.STEP` |
| LD06 雷达转接架 | 1 | `LD06雷达转接架.STEP` |
| 显示屏支架 | 1 | `显示屏支架.STEP` |

## 打印建议

- 推荐材料： PETG
- 填充密度：30%
- 壁厚：3 层
- 轮毂内圈为受力件，建议提高填充密度至 50%
