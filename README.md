## 文件结构

整个机器人项目被分成如下的几个部分，分别位于仓库不同目录下，内部有更详细的说明，读者可以按需查看：

- [`Guide`](Guide)：项目指南，包含项目介绍、功能说明、使用教程等文档
- [`Machine`](Machine)：机器人机械结构设计，包含机械设计文件和装配说明
- [`Code`](Code)：机器人软件部分，包含上位机和下位机模块的代码工程
- [`PCB`](PCB)：机器人电路设计，包含电路原理图和PCB设计文件
- [`Matlab`](Matlab)：算法仿真，包含模型建立、算法设计和仿真文件等

## 功能概览

- **轮足机器人控制** — 基于 STM32 的底层控制，支持ROS2上位机指令控制，具备前进/转向/跳跃等动作能力
- **SLAM/导航** — 集成 SLAM 建图和 Nav2 导航，实现自主导航和路径规划
- **ASR/TTS/KWS** — 本地部署语音识别、语音合成和唤醒词检测模型，实现离线语音交互
- **GIF 表情播放器** — 全屏 GIF 播放器，通过命名管道动态切换表情，实现情感化交互
- **Qwen3-4B LLM** — 在 NPU 上部署 Qwen3-4B-Instruct 模型，实现本地对话推理
- **Home Assistant 集成** — 将机器人功能集成到 Home Assistant 中，实现智能家居联动和语音控制
